User Instructions
  • Design Power Capacity: Provide a numerical value. and its unit is kilowatt.
  • Operation hour: Provide a numerical value between 0-24. and its unit is hours.
  • Depth of Discharge: Provide a numerical value between 0-100. It is in percentage.
  • Battery Efficiency: Provide a numerical value between 0-100. It is in percentage.
  • Battery Type: Choose from different types of batteries from the dropdown option list. Options provide cost per kilowatt-hour for the furthur calculations.
  • State: Choose from different states from the dropdown option list. This will provide state wise cost.

    • Click the Click to Calculate button for getting the cost estimation.

  • Required ESS Capacity: Energy Storage System capacity is calculated from the user input. Unit is Kilowatt-hours. The obtained value represent the required battery capacity for the project.
  • PV System Cost: Photovotaic sytem cost is based on data obtained from EnergySage. It provides PV system cost including components like inverter, master power panel (MPP), grid interface panel (GPP), etc.
  • ESS Cost: Energy storage system (ESS) cost is based on the required ESS capacity and the selected battery types. Cost varies from lead acid to lithium-ion batteries.

  • PV System Cost: PV system cost includes different components necessary for standalone operation
  • ESS Cost: ESS cost includes capital cost, power conversion system, balance of plant and construction and commisioning cost
